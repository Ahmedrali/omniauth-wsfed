# Omniauth WS-Fed

#### This gem is currently under construction... Expect an official v1 release by the the end of June 2012.

## Installation

Add this line to your application's Gemfile:

    gem 'omniauth-wsfed'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install omniauth-wsfed
